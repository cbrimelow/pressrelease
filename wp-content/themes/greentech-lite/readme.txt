== Description ==

GreenTech Lite is a beautiful and easy to use premium WordPress business theme for enviromental technical companies which can help you build on your own website with no time at all. This theme has helpful options for you to customize your website the easy way.

Built on the latest technology of WordPress, GreenTech Lite is lightweight, fast and optimized for all mobile phones.
